== Description ==

Gives the user the ability to add the archive of custom post types to a menu

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload entire cpt-archive-to-nav directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Within the Appearance -> Menus page, click on the Screen Options at the top to view other post types if your custom post type does not have it's own metabox.  Then click on the View All tab of the metabox and the top selection should be the Custom Post Type's Archive Page and can then be added to the menu and adjusted as needed.
